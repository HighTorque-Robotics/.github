# HighTorque Robotics

Hightorque致力于双足机器人和嵌入式智能解决方案的研发和制造，适用于消费者和科研场所。我们优先考虑自主创新和技术进步，专注于关键机器人部件的内部开发，如高性能电机、先进控制系统以及最先进的感知和运动控制算法。通过深度整合整个机器人生态系统的尖端技术，Hightorque旨在提供智能、多功能和可靠的机器人系统，从根本上改变自动化和人机协作的未来。
